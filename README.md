# python-conf-test
A repo to test if Python installation is working

## How to

Create and activate a virtualenv with 

```
python -m venv .venv
.\.venv\Scripts\activate
```

install requirements 

```
pip install -r requirements.txt
```

then launch it with

```
python main.py
```

Expected result:

```
Mean: 0.4902565533201336
```
